# DAC Presentation Site

Site static de prezentare pentru gama DAC, construit pe baza structurii și conținutului din `DAC_EU_structura_site.docx`.

## Conținut

- homepage DAC
- pagină de gamă
- categorii: camioane, vehicule speciale, autobuze electrice
- pagini dedicate pentru fiecare model din document
- pagini companie: Despre DAC, Istoria Brandului DAC, Service, Contact

## Fișiere principale

- `index.html`
- `site-data.js`
- `app.js`
- `styles.css`

## Rulare locală

Varianta recomandată în PowerShell:

```powershell
cd E:\DAC\repo
.\start-local.ps1
```

Apoi deschizi:

`http://127.0.0.1:8080`

Alternativ, poți porni direct:

```powershell
cd E:\DAC\repo
node .\server.js
```
